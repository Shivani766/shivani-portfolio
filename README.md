## 🧑‍💻 About The Project

This is a **Full Stack Portfolio Website** built using the **MERN Stack**. It is designed to showcase my projects, technical skills, and provide a seamless way for visitors to connect with me through a fully functional contact form backed by a server.

The project focuses on **performance, responsiveness, and clean UI/UX design**, making it accessible across all devices.

---

## ✨ Features

* 🎨 Modern and clean UI/UX design
* 📱 Fully responsive (mobile + desktop)
* 📂 Dynamic projects showcase section
* 📞 Contact form with backend integration
* ⚡ Optimized for fast performance
* 🔐 Secure API handling using Node.js & Express

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/portfolio.git
cd portfolio
```

### 2️⃣ Setup Frontend

```bash
cd frontend
npm install
npm run dev
```

### 3️⃣ Setup Backend

```bash
cd backend
npm install
node server.js
```
